# 0x03-git

At the end of this project you should be able to reproduce and
understand these commands:
```
$ git init
$ git clone <repo>
$ touch test
$ git add test
$ git commit -m "Initial commit"
$ git push origin main
$ git push -u origin main
$ git push
$ git pull
$ git merge <repo>
$ git branch <repo>
$ git checkout <repo>
$ git checkout -b <repo>
$ git status
$ git log
$ git add .

```
